# OPNSense Firewall
The firewall that I will use for the Proxmox environment is a virtual instance of OPNSense

## Setup
Created a new VM in Proxmox utilizing an image downloaded from the [OPNSense website][opnsense-image-url]

### Specs
Utilized the minimum specs [recommended][opnsense-specs] from OPNSense
- 1 CPU core
- 4 GiB memory
- 8 GiB storage
